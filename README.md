# 📦 Sistema de Gestão: Estoque, Vendas e Clientes

Este é um projeto acadêmico desenvolvido em linguagem C para a disciplina de [Nome da Disciplina] do 1º semestre de [Seu Curso]. O objetivo do sistema é gerenciar o fluxo de mercadorias, vendas e clientes de um estabelecimento através do terminal.

## 🚀 Funcionalidades do Sistema

### 🛠️ Controle de Estoque
* **Cadastro:** Inclusão de novos produtos.
* **Listagem:** Visualização completa dos itens cadastrados.
* **Remoção:** Exclusão de produtos do sistema.
* **Métricas financeiras:** Cálculo automático do valor total do estoque.
* **Relatórios Inteligentes:** * Relatório geral de estoque.
  * Relatório de **estoque baixo** (alerta para produtos que estão acabando).

### 🛒 Menu de Vendas
* **Inclusão de Vendas:** Registro de novas movimentações de saída.
* **Histórico:** Listagem detalhada de todas as vendas já realizadas.

### 👥 Gestão de Clientes & Tratamento de Dados
* **CRUD de Clientes:** Inclusão, listagem e remoção de clientes.
* **Validação de Dados:** * Bloqueio de campos obrigatórios em branco (como o nome).
  * Algoritmo para **validação real de CPF**.

## 💻 Tecnologias e Conceitos Aplicados
* **Linguagem:** C (Padrão ANSI)
* **Conceitos de Programação:** Structs (Estruturas), Vetores/Matrizes, Funções/Modularização, Validação de Strings e Lógica Condicional Avançada.
